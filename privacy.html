Privacy Policy

1. Data Collection

A Space Bot collects minimal data necessary for operation:

  • Server IDs for configuration settings
  • Basic usage statistics
  • Command usage data

2. Data Usage

Collected data is used solely for:

  • Providing bot services
  • Managing server-specific settings
  • Improving bot functionality

3. Data Sharing

We do not share or sell any collected data with third parties.

4. Data Retention

Command usage data is temporary and regularly cleared for privacy.
